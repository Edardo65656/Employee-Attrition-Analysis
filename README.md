# Employee Attrition Analysis

## Descripción del Proyecto
Este proyecto consiste en un dashboard interactivo desarrollado en Power BI para analizar la deserción de personal (Employee Attrition). El objetivo principal es identificar los factores clave que impulsan las renuncias y proporcionar a los responsables de la toma de decisiones información respaldada por datos para mejorar las estrategias de retención.

## Herramientas Utilizadas
* **Power BI Desktop:** Modelado de datos, diseño de interfaz y visualización.

## Insights y Preguntas de Negocio Resueltas
El análisis se centró en diagnosticar el "por qué" detrás de las renuncias, abordando tres áreas críticas:
1. **Deserción por Puesto:** Identificación de los roles con mayor volumen de salidas.
2. **Estancamiento Profesional:** Análisis de la relación directa entre los años sin recibir un ascenso (*Years Since Last Promotion*) y el aumento en la tasa de rotación.
3. **Carga de Trabajo y Satisfacción:** Evaluación del impacto que tienen las horas extras (*Overtime*) y el nivel de satisfacción laboral en la decisión de abandonar la empresa.

## Estructura de este Repositorio
* `Employee Attrition Analysis.pbix` - Archivo fuente de Power BI con el modelado de datos y visualizaciones.
* `WA_Fn-UseC_-HR-Employee-Attrition.csv` - Dataset original de IBM utilizado para el análisis.
* `README.md` - Documentación principal del proyecto.
